CSCE 242: Web Applications

Cade Phillips

CSCE 242 is a class in web technologies to support client-server computing. Implementation of client-server applications.

Assignments

  • Assignment 1: Basic HTML

    • Introduction to HTML
  • Assignment 2: Basic CSS

    • First look at CSS and how it works with HTML
  • Assignment 3: Page Layout

    • Learning how to use flexboxes
  • Assignment 4: Main Home Page CSS

    • Adding and using flexboxes on the main home page
  • Assignment 5: Recreate CSS Page

    • Recreating and replicating the UofSC CSS page
  • Assignment 6: JavaScript Buttons, Functions, and More

    • Learning how to use JavaScript in a webpage
  • Assignment 7: JavaScript Conditionals

    • Using conditionals for a color slider and changing the size of an image
  • Assignment 8: JavaScript Loops

    • Using a for loop to display stars, and checking the index of each star
  • Assignment 9: JavaScript Arrays

    • Using an array to display a description of a picture

Projects

  • Project 1: Topic Selection

    • Selecting the topic for the project and discussing the different pages
  • Project 2: Wireframes

    • Creating a wireframe prototype
  • Project 3: HTML and CSS

    • Recreating wireframes with HTML and CSS
  • Project 4: Colors, Pictures, and Text

    • Adding better text, colors, and pictures to part 3
